# ⬢ TaskFlow — Smart Task Manager

> **A blazing-fast, responsive Task Manager built with pure HTML, CSS & Vanilla JavaScript.**  
> No frameworks. No dependencies. Just clean, elegant code.

---

## 🚀 Live Demo

🔗 **[taskflow-burukalamanireethika.vercel.app](https://taskflow-burukalamanireethika.vercel.app)**

---

## ✨ Features

| Feature | Description |
|---|---|
| ➕ **Create / Edit / Delete** | Full CRUD for all your tasks |
| 🎯 **Priority Levels** | High, Medium, Low — color-coded cards |
| 📊 **Status Tracking** | To-Do → In Progress → Done |
| 🔍 **Real-time Search** | Instant filter as you type |
| 📅 **Due Dates** | Overdue alerts highlighted in red |
| 💾 **Auto-Save** | Persists with localStorage — no backend needed |
| 📱 **Mobile Responsive** | Works perfectly on all screen sizes |
| ⚡ **Zero Dependencies** | Pure HTML, CSS, JS — no npm, no build step |

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Flexbox, Grid, Animations, CSS Variables
- **Vanilla JavaScript (ES6+)** — No frameworks
- **Google Fonts** — Syne + DM Sans
- **localStorage** — Client-side persistence

## ⚡ Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/BurukalaManiReethika/taskflow.git

# 2. Navigate into the folder
cd taskflow

# 3. Open in browser — that's it!
open index.html
```

> No npm install. No build step. No config. Just open and run! 🎉

## 🚀 Deployment

### Vercel (Recommended)
1. Go to [vercel.com](https://vercel.com) → Sign in with GitHub
2. Import `taskflow` repository
3. Leave all settings default → Click **Deploy**
4. Live in 30 seconds! ✅

### GitHub Pages
1. Repo → **Settings** → **Pages**
2. Source: `main` branch → **Save**
3. URL: `https://burukalamanireethika.github.io/taskflow`
